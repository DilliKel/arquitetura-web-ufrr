# Teclado Virtual Responsivo

Projeto 2 da disciplina **DCC704 — Arquitetura e Tecnologias de Sistemas Web** (UFRR).

O objetivo deste trabalho é consolidar os conceitos sobre HTML, CSS e JavaScript, com foco na implementação de um teclado virtual responsivo, integrável a outros sistemas.

## Stack

HTML · CSS · JavaScript (vanilla)

## Como visualizar

Abra `index.html` no navegador. O teclado abre automaticamente ao focar em qualquer campo com a classe `use-keyboard-input`.
